# port_scanner
This Tool will scan an ip for open ports

Steps to run this tool:
   1. git clone https://github.com/krishpranav/port_scanner
   2. cd port_scanner
   3. sudo chmod 777 *
   4. python3 -m pip install -r requirements.txt
   5. python3 port_scanner.py
